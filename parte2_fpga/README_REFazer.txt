REFAZER O PROJETO NO QUARTUS RAPIDO

1. Extraia esta pasta.
2. Abra o Quartus.
3. File > Open Project.
4. Abra o arquivo trabaio.qpf.
5. Confira:
   Assignments > Settings > General > Top-level entity = DE2_115
6. Compile:
   Processing > Start Compilation.
7. Depois:
   Tools > Programmer.
8. Add File:
   output_files/trabaio.sof
9. Marque Program/Configure e clique Start.

Se o Quartus pedir para atualizar o projeto, aceite.
